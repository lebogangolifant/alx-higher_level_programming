Test Case Module: 100-matrix_mul.txt

This test case module verifies the functionality of the `matrix_mul`
function in the `100-matrix_mul.py` file.
It contains test cases to validate the multiplication of two matrices.

Test Cases:
- Test Case 1: Multiplication of two 2x2 matrices
    - Input: matrix_mul([[1, 2], [3, 4]], [[1, 2], [3, 4]])
    - Expected Output: [[7, 10], [15, 22]]
    - Description: The function should multiply the matrices and return the expected result.

- Test Case 2: Multiplication of a 1x2 matrix with a 2x2 matrix
    - Input: matrix_mul([[1, 2]], [[3, 4], [5, 6]])
    - Expected Output: [[13, 16]]
    - Description: The function should multiply the matrices and return the expected result.
